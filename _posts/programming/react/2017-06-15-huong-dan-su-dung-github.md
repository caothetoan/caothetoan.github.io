Commit and push
```
git init
git add --all
git commit -m "first commit"
git remote add origin https://github.com/caothetoan/Crawler-Python.git
git push -u origin master
```

update
```
git add --all
git commit -m "update commit"
git push -u origin master
```

clone
```
git clone https://github.com/caothetoan/Crawler-Python.git
git add --all
git commit -m "first commit"
git push -u origin master
```
